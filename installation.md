### Mise en oeuvre du Widget

> La mise en oeuvre du widget consiste à ajouter, dans une page GRIST éventuellement créée pour l'occasion dans le même document que la table à cartographier, un widget GRIST (ou vue personnalisée) de type URL en utilisant l'URL suivante : **https://lesagnic.github.io/grist-widget-explorer/widget**

### Configuration du Widget

Après avoir mise en oeuvre le widget Carte Facile et après l'avoir sélectionné, sa configuration se fait en deux étapes au sein de l'onglet **Vue** se trouvant dans la partie droite de la page :
* Autoriser le widget à lire les données de la table ;
* Mettre en correspondant les colonnes de la table avec les propriétés **Longitude**, **Latitude** et **Libellé** du widget.

> Chaque ligne disposant de données cartographiques valides est alors par défaut représentée sur le fond Carte Facile avec ce symbole : <img src="widget/marker.png" alt="Marker" height="16"/>.
