Il faut d'abord disposer d'une **table GRIST disposant a minima de**:
* deux colonnes contenant un couple de coordonnées géographiques (**longitude** et **latitude**) idéalement exprimées dans le système ETRS89 ;
* une colonne pouvant servir de **libellé**.

> Les lignes de la table ne contenant pas de valeur pour une de ces colonne seront ignorées/

Il faut également disposer d'une page, dans le même document GRIST que la table, dans laquelle intégrer le **widget Carte Facile**.
